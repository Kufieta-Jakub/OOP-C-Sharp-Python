# ConsoleApp11 – Zbiór zadań z C#

Prosty program konsolowy w C# zawierający trzy zadania z podstaw programowania. Po uruchomieniu użytkownik wybiera zadanie z menu.

## Zadania

### 1. Kalkulator

Użytkownik podaje dwie liczby całkowite oraz operator (`+`, `-`, `*`, `/`). Program oblicza i wyświetla wynik.

**Przykład:**
Podaj pierwsza liczbe: 10
Podaj druga liczbe: 20
Podaj operator: +
Wynik to: 30

### 2. Konwerter temperatur

Konwersja między skalą Celsjusza a Fahrenheita. Użytkownik wybiera kierunek (`C` lub `F`), podaje temperaturę, a program przelicza ją według wzorów:

- °F = °C × 1.8 + 32
- °C = (°F − 32) / 1.8

**Przykład:**
Wybierz kierunek konwersji (C lub F): C
Podaj temperaturę: 0
0°C = 32°F

### 3. Średnia ocen ucznia

Użytkownik podaje liczbę ocen, a następnie wpisuje je kolejno (skala 1–6). Program oblicza średnią i sprawdza, czy uczeń zdał (średnia ≥ 3.0).

**Przykład:**
Podaj liczbę ocen: 4
Podaj ocenę 1: 5
Podaj ocenę 2: 3
Podaj ocenę 3: 4
Podaj ocenę 4: 2
Średnia: 3.50
Uczeń zdał.
